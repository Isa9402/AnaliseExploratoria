# AnaliseExploratoria
 Commit é uma ferramenta usada no Git para integrar as mudanças que você fez no seu arquivo para o seu repositório no GitHub

Resultados Gráficos:

O histograma da variável idade mostrou a distribuição etária da amostra, com maior concentração entre 50
e 60 anos. A escala foi ajustada para intervalos de 10 anos no eixo X e frequência de 5 em 5 no eixo Y.

O boxplot da variável tempo_preso indicou a dispersão e possíveis outliers no tempo de prisão dos 
indivíduos. Foi possível observar que a maioria dos valores se concentra entre o 50 e 70 meses,
com alguns casos acima da mediana.

O boxplot da variável score_periculosidade por escolaridade permitiu comparar o comportamento do score
entre diferentes níveis de escolaridade. Observou-se variação na mediana e na amplitude entre os grupos, 
sugerindo diferenças de perfil conforme o grau de instrução.

O gráfico de barras da variável reincidente mostrou a proporção de indivíduos reincidentes
e não reincidentes, evidenciando o percentual predominante onde a maioria dos detentos não é reincidente.
